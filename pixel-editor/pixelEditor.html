Group 16's- Pixel Sprite Editor

Instructions:
-The default mouse color is 'black'
-Click on any of the pixels in the grid to change their color
-Click on the 'white' or 'black' boxes above the grid to switch the color of the mouse
-The "Convert to Hex" converts the 15 x 8 sprite into 15 hexadecimal values that can be used to create CHIP-8 sprites
-Reload the page to reset all pixel locations or to erase the previous hex values
